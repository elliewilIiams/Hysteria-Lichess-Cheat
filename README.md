#### Hysteria Lichess.org Cheat
## Стек:
Полноценный Fullstack Local софт, не зависящий от внешних API.
- **Backend**
    -  *Python3*
        - *Flask* (http сервер)
        - *python-chess* (работа с FEN образами)
        - *Stockfish* (сам шахматный движок для принятия решений)
- **Frontend**
    - **TypeScript**
        - *esbuild* (для сбора .ts -> .js)
        - *DOM parsing* (парсинг позиции на доске)

## Установка
- Скачайте или клонируйте репозиторий в необходимую вам папку
- Файл движка (stockfish-windows-x86-64.exe) необходимо перенести в любую папку, скопировать ее путь и указать в *./app/server.py* (по дефолту C:/stockfish/stockfish-windows-x86-64.exe)
- Запустить бекенд
```bash
cd {путь}
pip install -r requirements.txt
python app/server.py
```
- Собрать расширение
```bash
cd {путь}
npm install
npm run build
```
- в строку сhrome *chrome://extensions* -> Developer mode → Load unpacked → выбрать папку `extension/`.

## Использование

1. Запустить `python app/server.py`
2. Открыть партию на Lichess.org
3. Расширение автоматически анализирует позицию и рисует стрелки лучших ходов по разным уровням, про них ниже
4. В меню расширения есть возможность отключить анализ

## Уровни анализа

Расширение отрисовывает 4 уровня хода, которые бекенд успевает просчитать за выделенные *300 мс*
На деле за этот промежуток времени просчет доходит от 12 (тяжелая позиция) до 25 ходов вперед (начало игры/легкая позиция)
- Уровни
    - 🟢 зелёный - Самый лучший ход по мнению движка Stockfish, самый яркий
    - 🟡 оранжевый и 🔵 синий одинаковые по уровню
    - 🟣 фиолетовый - Худший ход
