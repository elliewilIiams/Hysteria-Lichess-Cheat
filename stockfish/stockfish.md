#### Как установить Stockfish чтобы чит мог подключится к нему

Возьмите файл stockfish-windows-x86-64.exe находящийся по соседству и закиньте в нужную вам папку, скопируйте путь и вставьте в файл server.py

#### How to install Stockfish so the cheat can connect to it

Take the stockfish-windows-x86-64.exe file located alongside the project and move it to any folder you want. Copy the path to the executable and paste it into the server.py file.
